### Running the application

- Linux/Mac

```
./mvnw spring-boot:run
```

- Windows

```
mvnw.cmd spring-boot:run
```

- IDE

In your favourite java IDE import the project and run/debug the `src/main/java//es/manolo/demo_jug/DemoJugApplication.java` class
